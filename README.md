
# Blades in the Dark - 한국어 (Korean)

이 모듈은 Foundry VTT에서 사용되는 [Blades in the Dark 시스템](https://github.com/megastruktur/foundryvtt-blades-in-the-dark)을 위한 한국어 번역 모듈입니다.

[justinross의 얼터너티브 캐릭터 시트](https://github.com/justinross/foundry-bitd-alternate-sheets)도 지원합니다.

## 설치 및 사용 방법

* **Blades in the Dark** 시스템을 설치하세요.
* [Babele](https://foundryvtt.com/packages/babele) 모듈을 설치하세요.
* 본 한국어 번역 모듈을 설치하세요.
* 월드 내 설정 패널에서 위 모듈들을 활성화하세요.
* 설정(Configure Settings) 메뉴에서 언어를 **한국어(Korean)**로 변경하세요.
